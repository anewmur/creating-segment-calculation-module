# Prototype

Директория `prototype/` содержит локальный прототип для ручного запуска расчёта сегментов и визуализации полигонов через SVG.

## Запуск

```bash
python -m prototype.run_prototype
```

## Результат

HTML-файл с визуализацией сохраняется в `data/result.html` в корне проекта и затем открывается в браузере.
